Class Thoughts

Collection of processes and branches to execute with middleware and state. Will use global path by default, but can be replaced with custom set. Sequence arrays define orders of named processes, to run consecutively. Default sequences are receive and dispatch to process incoming/outgoing. Each process may have a validate method to run before processing and an action method to run after. Validate returning false will skip the process.
